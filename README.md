# bridge
[http bridge](https://github.com/ton-connect/docs/blob/main/bridge.md) for tonconnect 2.0:

## requirements
- golang 1.18
- postgres

## how to  install
- git clone https://github.com/ton-connect/bridge
- cd bridge
- go build ./ 
- go run bridge
